# Context 

Xiangqi, also known as Chinese Chess, is one of the most popular board game in China that is played by millions of people each day. More information on the history of the game and its rules can be found from the [wikipedia page](https://en.wikipedia.org/wiki/Xiangqi)


# Content

What's inside is more than just rows and columns. Make it easy for others to get started by describing how you acquired the data and what time period it represents, too.

The dataset contains 10,000 game logs of xiangqi played on [playOK.com](http://www.playok.com/en/xiangqi/), scraped off playOK API with Python. In particular, the games have ID number between 57380690 and 57390689. There are two files in the dataset:

1. gameinfo.csv which contains players information and game result in
    + gameID
    + game_datetime
    + blackID
    + blackELO
    + redID
    + redELO
    + winner
    
2. moves.csv which contains game moves in
    + gameID
    + turn: at which turn of the game the move was made
    + side
    + move: recorded with the WXF notation. Explaination can be found at [XQinEnglish](http://www.xqinenglish.com/intro_to_xq_notations.html)
